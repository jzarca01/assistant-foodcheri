# assistant-foodcheri

Un plugin pour suivre l'état de sa commande Foodchéri

## Configuration

```json
{
  "foodcheri": {
    "email": "",
    "password": ""
  }
}
```

## Utilisation

- [Suivre ma commande Foodchéri](https://ifttt.com/applets/88257417d-if-you-say-ou-en-est-mon-foodcheri-then-push-a-note)
